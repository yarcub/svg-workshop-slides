SVG and HTML5

SVG and HTML5

SVG and HTML5

in a nutshell

  • Open Standard (W3C)
  • Describes 2D vector images in a declarative form
  • Full DOM support
    • Scripting
    • CSS Styling
    • Event Model
  • Modern Browser Support (plugins free)
    • Mobile included (kinda)

SVG PRIMITVES

Rectangle

        <rect x="350" y="50" height="100" width="200" fill="yellow" stroke="black"/>
      </svg>

Circle and Ellipse


      <svg>
       <!-- center in cx,xy with radius r -->
       <circle cx="250" cy="150" r="100" fill="yellow" stroke="black"/>
       <!-- center in cx,xy with radius rx,ry -->
       <ellipse cx="650" cy="150" rx="100" ry="50" fill="red" stroke="black"/>
      </svg>

Line and Polyline


      <svg>
       <!-- from point x1,y1 to point x2,y2-->
       <line x1="0" y1="0" x2="1000" y2="250" stroke-width="5" stroke="#000"/>
       <!-- from point x1,y1 to point xn,yn passing by xi,yi -->
       <polyline points="0,0 300,100 600,300 800,250" 
                 stroke-width="5" fill="none" stroke="red" />
      </svg>

Polygon


      <svg>
       <!-- from point x1,y1 to point x1,y1 passing by xi,yi (auto close)-->
       <polygon points="0,0 300,100 600,300 800,250" 
                 stroke-width="5" fill="rgb(255,0,0,.5)" stroke="red" />
      </svg>

Path

Path

Sequential group of commands

  • Move to
  • Line to
  • Close path
  • Bézier (cubic or quadratic) curve to
  • Elliptical arc to

Path


      <svg>
       <!-- 
        Move to 200,300
        Quadratic Bézier to 400,50 with control point 600,300
        Smooth Quadric Bézier to 1000,300 (mirror last control point)
        -->
       <path d="M200,300 Q400,50 600,300 T1000,300"
                fill="none" stroke="red" stroke-width="5"  />
      </svg>

Text & Paths

       <svg>
        <def>
          <path id="curve" d="M100,100 Q350,0 450,100 T800,100"
              fill="none" stroke="rgba(255,0,0,0.5)" stroke-width="2"  />
        </def>
        <text>
          <textPath id="myText" xlink:href="#curve">
            Look, i'm text following a bézier curve! Select me =)
          </textPath> 
        </text>
        <use xlink:href="#curve"></use>
      </svg>

Scripting

Scripting

User interaction

        var rect = document.querySelector("#foo");
        var label = document.querySelector("#label");
        
        rect.addEventListener('mouseover', function(event){
          label.textContent = "Over";
          //SVG DOM, faster
          rect.width.baseVal.value = 500; 
        });

        rect.addEventListener('mouseout', function(event){
          label.textContent = "Out";
          //HTML DOM, slower, needs to convert string to int
          rect.setAttribute("width", "200"); 
        });

    • Version Control
    • Semantic + metadata
  • Bandwidth
    • Small filesize
    • Runtime changes
    • Gzip encoded (.svgz)
